Инструкция по запуску и работе с сайтом
=======================================

1. Установка зависимостей:
   npm install
   # или
   yarn install
   # или
   pnpm install

2. Запуск сервера разработки:
   npm run dev
   # или
   yarn dev
   # или
   pnpm dev

3. Сборка проекта:
   npm run build
   # или
   yarn build
   # или
   pnpm build

4. Запуск собранного проекта:
   npm run preview
   # или
   yarn preview
   # или
   pnpm preview

5. Проверка кода:
   npm run lint
   # или
   yarn lint
   # или
   pnpm lint

Дополнительные команды
=======================================

1. Очистка кэша:
   npm run clean
   # или удаление папок dist и node_modules вручную

2. Обновление зависимостей:
   npm update
   # или
   yarn upgrade
   # или
   pnpm update

3. Запуск тестов (если настроены):
   npm run test
   # или
   yarn test
   # или
   pnpm test

Решение возможных проблем
=======================================

1. Если возникают ошибки при установке:
   - Удалить node_modules и package-lock.json
   - Выполнить npm cache clean --force
   - Повторить npm install

2. Если не запускается сервер разработки:
   - Проверить, что порт 5173 свободен
   - Попробовать запустить с другим портом:
     npm run dev -- --port 3000

3. Проблемы с зависимостями:
   - Проверить версии в package.json
   - Убедиться, что установлена правильная версия Node.js
   - Попробовать установить зависимости с флагом --legacy-peer-deps

Важные замечания
=======================================

1. Для корректной работы требуется Node.js версии 16.0.0 или выше

2. Рекомендуется использовать последние версии браузеров:
   - Chrome
   - Firefox
   - Safari
   - Edge

3. Для разработки рекомендуется использовать VS Code с расширениями:
   - ESLint
   - Prettier
   - Vite
   - React Developer Tools
