Используемые технологии и их версии
====================================

Основные технологии:
------------------
1. React 18.2.0
2. Vite 5.0.0 (сборщик проекта)
3. React Router DOM 6.20.0
4. CSS Modules

Стилизация:
----------
1. CSS Variables (Custom Properties)
2. Flexbox
3. CSS Grid
4. Media Queries для адаптивного дизайна

Языки программирования:
---------------------
1. JavaScript (ES6+)
2. JSX
3. CSS3
4. HTML5

Инструменты разработки:
---------------------
1. ESLint 8.54.0
2. Prettier 3.1.0
3. Node.js 16.0.0 или выше
4. npm/yarn/pnpm (пакетные менеджеры)

Браузерные API:
-------------
1. Fetch API
2. LocalStorage API
3. History API
4. Web Animations API

Особенности и функционал:
-----------------------
1. Компонентный подход (React Components)
2. Хуки React (useState, useEffect, useContext)
3. Роутинг (React Router)
4. Асинхронные запросы
5. Управление состоянием
6. Условный рендеринг
7. Обработка событий
8. Интерактивные элементы UI

Форматирование и локализация:
---------------------------
1. Intl API для форматирования дат
2. Русская локализация (ru-RU)

Планируемые технологии для расширения:
-----------------------------------
1. Redux/MobX для управления состоянием
2. TypeScript для типизации
3. Jest для тестирования
4. Storybook для документации компонентов
5. Firebase/MongoDB для базы данных
6. Express.js для backend API
